## `$ about`

This repository tracks my ongoing **Data Structures & Algorithms** practice — problem-solving across arrays, strings, trees, graphs, dynamic programming, and core CS fundamentals. Solutions are organized by topic, with a focus on clarity, complexity analysis, and clean implementation.

Part of my **BS Artificial Intelligence** coursework at the **University of Karachi**, where DSA is a core subject alongside OOP, Discrete Structures, and Digital Logic Design.
